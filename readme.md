## 1° Download the project and enter it:

```bash
git clone git@github.com:pauloteixeira01/adonis-api.git

```

```bash
cd adonis-api
```

## 2° Download dependencies:

```bash
npm install
```

If you have some problem, run:

```bash
npm install --ignore-platform
```

## 3° Run the project:

```bash
npm run dev
```
